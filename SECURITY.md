# Security Policy

## Supported versions
Nexus is pre-1.0; only the latest `main` and the current TestFlight build are supported.

## Reporting a vulnerability
**Do not open a public issue for security problems.** Use GitHub's private reporting:
**Security → Advisories → Report a vulnerability** on this repo. If you can't, email
`nexuspersonalapp@gmail.com` with details and a proof-of-concept if possible.

You'll get an acknowledgement within a few days. Please give a reasonable window to fix
before any public disclosure.
